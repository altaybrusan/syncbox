MicroC command map

A set of functions that maps the MicroC UART commands into more readable form.

Kernel.c needs to write into serial ports. To make the Kernel.c more clear a set of SendTo macros are defined.

Note
Instead of sending whole message at once, using MikroC Uartx_Write_Text, charachter based transmission is prefered. In practice, Uartx_Write_Text addes some not-wanted charachters to the end of transmitted message. For example, MeSsAgEü may appear instead of !MeSsAgE.

Detailed Description

Contains global configuration and settings of connections. Each device type (generator, actuator,etc.) has a special set of connection parameters. The ConnectionMap.h stores all these parameters. The
