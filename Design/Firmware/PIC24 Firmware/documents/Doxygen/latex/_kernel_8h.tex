\subsection{Detailed Description}
public interface of the x\+P\+O\+ST kernel. 

The kernel header files include kernel\textquotesingle{}s public methods,structures and enums. This file can be seen as the {\itshape public} {\itshape interface} of the kernel. The device A\+P\+Is and instrument can include this header file and then call the kernels services.
