\subsection{Detailed Description}
Contains global configuration and settings of connections. Each device type (generator, actuator,etc.) has a special set of connection parameters. The \hyperlink{_connection_map_8h}{Connection\+Map.\+h} stores all these parameters. The \begin{DoxySeeAlso}{See also}
\hyperlink{_kernel_8c}{kernel.\+c} uses the connection parameters to setup the microntroller. 
\end{DoxySeeAlso}
\begin{DoxyNote}{Note}
The A\+PI\textquotesingle{}s normally do not use \hyperlink{_connection_map_8h}{Connection\+Map.\+h} unless for debugging. 
\end{DoxyNote}
